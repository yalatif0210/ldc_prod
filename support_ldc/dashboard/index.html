Comment le client accède au bot ?

  • 1
    Le client scanne ce QR code ou clique sur le lien t.me/… partagé.
  • 2
    Telegram s'ouvre directement sur la conversation avec le bot.
  • 3
    Le client tape /start — le bot guide en 5 étapes.
  • 4
    Si un agent est disponible → assignation immédiate + notification client.
  • 5
    Sinon → ticket en file d'attente, client notifié à la prise en charge.
ℹ CONFIGURATION AGENT
Pour recevoir les tickets, chaque agent doit avoir son Telegram Chat ID configuré dans son profil. L'agent peut l'obtenir en envoyant /start au bot @userinfobot sur Telegram.
